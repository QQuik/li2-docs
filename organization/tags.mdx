# Tags

Organize your touchpoints with custom tags to streamline campaign management, enable powerful filtering, and improve workflow organization. Tags provide a flexible way to categorize and find your marketing assets quickly.

### Tag Limits by Tier

<Note>
  **Tag Limits**: Free tier users can create up to **20 tags per account**. Premium tier users can create up to **100 tags per account**. All users can assign up to **5 tags per touchpoint**.
</Note>

## Accessing Tags Management

### Navigating to Tags

<Steps>
  <Step title="Access tags section">
    1. Go to the main navigation menu
    2. Click on **Tag** in the sidebar
    3. You'll see the tags management interface with all your existing tags
  </Step>
  <Step title="Understanding the tags interface">
    The tags management screen shows:

    - **Color indicator** for visual identification
    - **Tag name** for reference
    - **Touchpoint number** showing how many touchpoints use each tag
    - **Actions** for editing and deleting tags
  </Step>
</Steps>

### Search and Filter Options

**Available Search Functions**:

- **Search by name**: Find tags by typing tag names
- **Search by created at**: Filter tags by creation date
- **View options**: Customize how tags are displayed in the interface

## Creating New Tags

### Tag Creation Process

<Steps>
  <Step title="Start creating a tag">
    1. Click the **"\+ Create Tag"** button in the tags interface
    2. The **Create Tag** dialog opens
    3. You'll see fields for tag name, color selection, and description

       ![Organize With Tags Pn](/images/Organize-with-tags.png)
  </Step>
  <Step title="Configure tag details">
    **Required Information**:

    - **Name Tag**: Enter a descriptive name for your tag
    - **Color**: Choose a color for visual identification
    - **Description**: Add optional description for context (recommended)

    **Tag Naming Best Practices**:

    - Use consistent naming conventions
    - Keep names concise but descriptive
    - Consider your campaign workflow and terminology
  </Step>
  <Step title="Save your tag">
    1. Review your tag configuration
    2. Click **"Create Tag"** to save
    3. Your new tag appears in the tags list immediately
    4. Tag is now available for assignment to touchpoints

       ![Configure Tag Details Pn](/images/Configure-tag-details.png)
  </Step>
</Steps>

## Managing Existing Tags

### Tag Details and Editing

<Steps>
  <Step title="Access tag details">
    1. Click on any tag name in the tags list
    2. Opens the **Tag Detail** screen
    3. Shows complete tag information and associated touchpoints
    4. Displays creation date and creator information
  </Step>
  <Step title="Edit tag information">
    **Editable Elements**:

    - **Name**: Update tag name as needed
    - **Color**: Change color for better organization
    - **Description**: Modify or add description text

    **Editing Process**:

    1. Modify any field in the tag detail screen
    2. Click **"Save Tag"** to apply changes
    3. Changes apply immediately across all touchpoints using the tag

       ![Tag Ssignments Pn](/images/Tag-ssignments.png)
  </Step>
</Steps>

### Managing Tag Assignments

<AccordionGroup>
  <Accordion title="Adding touchpoints to tags">
    **From Tag Detail Screen**:

    1. Click **"\+ Add Touchpoint"** in the tag detail view
    2. Search and select touchpoints from the **Add Touchpoint** dialog
    3. Use checkboxes to select multiple touchpoints
    4. Click **"Add Touchpoint"** to assign the tag

    **Bulk Assignment Benefits**:

    - Quickly organize multiple touchpoints under one tag
    - Efficient campaign setup and organization
    - Streamlined workflow for campaign management
  </Accordion>
  <Accordion title="Viewing tagged touchpoints">
    **Tag Detail View Shows**:

    - All touchpoints currently assigned to the tag
    - Touchpoint names and URLs
    - Creation dates for reference
    - Direct access to touchpoint management

    **Touchpoint Management**:

    - Click on any touchpoint to view or edit
    - See at a glance which touchpoints belong to specific campaigns
    - Quick access to touchpoint analytics and performance
  </Accordion>
</AccordionGroup>

### Deleting Tags

<Warning>
  **Tag Deletion**: When you delete a tag, it's removed from all touchpoints that were using it. The touchpoints themselves remain unchanged, but they will no longer have that tag assignment.
</Warning>

<Steps>
  <Step title="Delete a tag">
    1. Navigate to the tag detail screen
    2. Click the **"Delete Tag"** button (red button)
    3. Confirm deletion when prompted
    4. Tag is removed from all touchpoints and deleted permanently

       ![Delete Tag Pn](/images/delete-tag.png)
  </Step>
</Steps>

## Using Tags for Organization

### Tagging Strategies

<CardGroup cols={2}>
  <Card title="Campaign-Based Tagging" icon="calendar" color="#10B981">
    **Examples**: "summer-2025", "product-launch-q4", "holiday-campaign"

    Perfect for organizing touchpoints by specific marketing campaigns or time periods.
  </Card>

  <Card title="Audience-Based Tagging" icon="users" color="#10B981">
    **Examples**: "enterprise", "sme", "consumer", "b2b", "b2c"

  </Card>
  <Card title="Status-Based Tagging" icon="flag" color="#10B981">
    **Examples**: "active", "testing", "archived", "draft", "approved"

    Useful for workflow management and campaign lifecycle tracking.
  </Card>
</CardGroup>

### Multi-Tag Strategy

<AccordionGroup>
  <Accordion title="Combining multiple tags">
    **Strategic Tag Combinations** (up to 5 tags per touchpoint):

    - Campaign \+ Channel: "summer-2025" \+ "social-media"
    - Audience \+ Status: "enterprise" \+ "active"
    - Campaign \+ Audience \+ Channel: "product-launch" \+ "b2b" \+ "email"

    **Benefits of Multi-Tagging**:

    - More precise filtering and organization
    - Complex campaign management capabilities
    - Detailed analytics and reporting options
    - Flexible workflow organization
  </Accordion>
</AccordionGroup>

## Tags in Touchpoint Assignment

### Adding Tags to Touchpoints

<Steps>
  <Step title="During touchpoint creation">
    1. When creating a new touchpoint, find the **Tag** field
    2. Start typing to see autocomplete suggestions
    3. Select existing tags or create new ones
    4. Add up to 5 tags per touchpoint
    5. Tags are assigned when the touchpoint is created
  </Step>
  <Step title="Adding tags to existing touchpoints">
    1. Edit any existing touchpoint
    2. Find the **Tag** field in the touchpoint details
    3. Use autocomplete to add tags
    4. Save changes to apply tag assignments
  </Step>
</Steps>

### Autocomplete and Tag Suggestions

<Note>
  **Smart Tag Input**: The tag field includes autocomplete functionality that suggests existing tags as you type. You can select from existing tags or create new ones directly in the input field.
</Note>

**Autocomplete Benefits**:

- **Consistency**: Reduces duplicate or similar tags
- **Efficiency**: Quickly find and assign existing tags
- **Discovery**: See all available tags while typing
- **Flexibility**: Create new tags when existing ones don't fit
