## Thêm Thẻ cho một link

Thẻ có thể được thêm ngay hoặc sau khi **Short link** được tạo. Bạn có thể thêm thẻ cho bất kỳ link nào trong tài khoản từ trang **Short link**.

### **Cách thêm thẻ vào một link**:

1. **Đăng nhập** tài khoản tại 👉 [app.li2.ai](http://app.li2.ai).
2. Vào mục **Short link**.
3. Chọn link cần gắn thẻ.
4. Trong khu vực hiển thị tags → nhấp **\+ Thêm thẻ**.
5. Chọn tag có sẵn hoặc nhập tên tag mới → nhấn **Enter** để [lưu.](http://lưu.Tr)

Tag mới sẽ hiển thị ngay trên link đó.

### Thêm Thẻ cho nhiều link cùng lúc

Bạn có thể thêm thẻ cho nhiều link cùng lúc để tiết kiệm thời gian.

**Cách thêm thẻ cho nhiều link**:

1. **Đăng nhập** tài khoản tại 👉 [app.li2.ai](http://app.li2.ai).
2. Trong menu bên trái, chọn **Thẻ (Tag)**.
3. Tìm **Thẻ** bạn muốn thêm nhiều link → nhấp vào biểu tượng **✏️ Edit**.
4. Nhập **\+ Thêm Touchpoints**  → Tích chọn ô kiểm bên cạnh các link bạn muốn thêm thẻ.
5. Sau đó nhấn **Thêm Touchpoints** để lưu.

👉 Rất tiện lợi khi bạn muốn nhóm toàn bộ link của một chiến dịch.

---

### Xóa Thẻ khỏi một Link

**Cách gỡ thẻ khỏi một link**:

1. Đăng nhập vào tài khoản [Li2.ai](http://Li2.ai) tại [app.li2.ai](http://app.li2.ai).
2. Nhấp vào \*\*Short link \*\*trong thanh bên trái.
3. Tìm link bạn muốn gỡ thẻ và nhấp vào biểu tượng chỉnh sửa (hình bút chì).
4. Nhấp vào **X** trên thẻ bạn muốn gỡ.
5. Nhấp **Lưu** khi hoàn tất.

---

### Lọc Liên kết Theo Thẻ

Sau khi đã thêm thẻ cho liên kết, bạn có thể lọc danh sách liên kết dựa trên các thẻ này để quản lý dễ dàng hơn.

**Cách lọc theo thẻ**

1. Đăng nhập vào tài khoản [Li2.ai](http://Li2.ai) tại [app.li2.ai](http://app.li2.ai).
2. Nhấp vào **Short link** trong thanh bên trái.
3. Nhấp vào **Thêm Bộ Lọc**.
4. Chọn các thẻ bạn muốn sử dụng làm bộ lọc, sau đó nhấp **Áp Dụng**.

   <Note>
     Các thẻ giúp bạn tổ chức quản lý chiến dịch một cách khoa học, ví dụ: tạo thẻ như `chien-dich-mua-he`, `mang-xa-hoi`, hoặc `email-marketing`để phân loại liên kết theo kênh hoặc mục tiêu.
   </Note>

Bắt đầu sử dụng thẻ ngay hôm nay để quản lý các liên kết của bạn hiệu quả hơn với [Li2.ai](http://Li2.ai)!

## Managing Existing Tags

### Tag Details and Editing

<Steps>
  <Step title="Access tag details">
    1. Click on any tag name in the tags list
    2. Opens the **Tag Detail** screen
    3. Shows complete tag information and associated touchpoints
    4. Displays creation date and creator information
  </Step>
  <Step title="Edit tag information">
    **Editable Elements**:

    - **Name**: Update tag name as needed
    - **Color**: Change color for better organization
    - **Description**: Modify or add description text

    **Editing Process**:

    1. Modify any field in the tag detail screen
    2. Click **"Save Tag"** to apply changes
    3. Changes apply immediately across all touchpoints using the tag
  </Step>
</Steps>

### Managing Tag Assignments

<AccordionGroup>
  <Accordion title="Adding touchpoints to tags">
    **From Tag Detail Screen**:

    1. Click **"+ Add Touchpoint"** in the tag detail view
    2. Search and select touchpoints from the **Add Touchpoint** dialog
    3. Use checkboxes to select multiple touchpoints
    4. Click **"Add Touchpoint"** to assign the tag

    **Bulk Assignment Benefits**:

    - Quickly organize multiple touchpoints under one tag
    - Efficient campaign setup and organization
    - Streamlined workflow for campaign management
  </Accordion>
  <Accordion title="Viewing tagged touchpoints">
    **Tag Detail View Shows**:

    - All touchpoints currently assigned to the tag
    - Touchpoint names and URLs
    - Creation dates for reference
    - Direct access to touchpoint management

    **Touchpoint Management**:

    - Click on any touchpoint to view or edit
    - See at a glance which touchpoints belong to specific campaigns
    - Quick access to touchpoint analytics and performance
  </Accordion>
</AccordionGroup>

### Deleting Tags

<Warning>
  **Tag Deletion**: When you delete a tag, it's removed from all touchpoints that were using it. The touchpoints themselves remain unchanged, but they will no longer have that tag assignment.
</Warning>

<Steps>
  <Step title="Delete a tag">
    1. Navigate to the tag detail screen
    2. Click the **"Delete Tag"** button (red button)
    3. Confirm deletion when prompted
    4. Tag is removed from all touchpoints and deleted permanently

       ![Delete Tag Pn](/images/delete-tag.png)
  </Step>
</Steps>

## Tags in Touchpoint Assignment

### Adding Tags to Touchpoints

<Steps>
  <Step title="During touchpoint creation">
    1. When creating a new touchpoint, find the **Tag** field
    2. Start typing to see autocomplete suggestions
    3. Select existing tags or create new ones
    4. Add up to 5 tags per touchpoint
    5. Tags are assigned when the touchpoint is created
  </Step>
  <Step title="Adding tags to existing touchpoints">
    1. Edit any existing touchpoint
    2. Find the **Tag** field in the touchpoint details
    3. Use autocomplete to add tags
    4. Save changes to apply tag assignments
  </Step>
</Steps>

### Autocomplete and Tag Suggestions

<Note>
  **Smart Tag Input**: The tag field includes autocomplete functionality that suggests existing tags as you type. You can select from existing tags or create new ones directly in the input field.
</Note>

**Autocomplete Benefits**:

- **Consistency**: Reduces duplicate or similar tags
- **Efficiency**: Quickly find and assign existing tags
- **Discovery**: See all available tags while typing
- **Flexibility**: Create new tags when existing ones don't fit
