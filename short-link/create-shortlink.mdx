## Rút gọn link trên [Li2](http://Li2.ai)

Để tạo rút gọn link trên Li2, bạn cần đăng nhập vào tài khoản của mình. Nếu chưa có tài khoản, hãy truy cập vào trang chủ và tạo tài khoản.

Hướng dẫn này cung cấp các bước để rút gọn link thông qua ứng dụng web [Li2.ai](http://Li2.ai) hoặc trên trang chủ [app.li2.ai](http://app.li2.ai):

- Tạo & Rút gọn link trên ứng dụng web[ Li2.ai](http://Li2.ai)
- Tạo & Rút gọn link trên trang chủ [app.Li2.ai](http://app.li2.ai)

### Link rút gọn là gì

Link rút gọn giúp biến một URL dài thành một URL ngắn gọn, dễ nhớ và dễ chia sẻ.

- Ví dụ:
  - URL gốc: `https://www.yourwebsite.com/product/abc123`
  - Link ngắn: `https://li2.ai/abc`
  - Link thương hiệu: `https://yourbrand.com/summer-sale`

Nếu bạn đã có tài khoản [Li2](http://Li2.ai), hãy làm theo các bước sau để tạo link rút gọn:

## Cách rút gọn link trang chủ [Li2](http://Li2.ai)

Nếu bạn chưa đăng nhập hoặc muốn thử nhanh:

1. Truy cập [li2.ai](http://li2.ai)
2. Tìm trường **Rút Gọn Link** trên trang chủ.
3. Dán **URL đích**, sau đó nhấp vào **Nhận liên kết miễn phí.**
4. Sao chép link rút gọn đã được tạo (Lưu ý: bạn cần đăng nhập để lưu, tùy chỉnh, hoặc theo dõi liên kết).

<Tip>
  Để sử dụng các tính năng nâng cao như tùy chỉnh URL, tạo mã QR cho link, hoặc phân tích chi tiết, bạn cần đăng nhập vào tài khoản [Li2.ai](http://Li2.ai).
</Tip>

Sẵn sàng bắt đầu? Đăng ký miễn phí tại li2.ai hoặc tham khảo thêm tại Doc.li2.ai để khám phá cách tối ưu hóa chiến dịch tiếp thị của bạn với [Li2.ai](http://Li2.ai)!
