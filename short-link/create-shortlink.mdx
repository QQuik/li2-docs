## Rút gọn link trên Li2

Để tạo rút gọn link trên Li2, bạn cần đăng nhập vào tài khoản của mình. Nếu chưa có tài khoản, hãy truy cập vào trang chủ và tạo tài khoản.

Cách rút gọn link tại trang web \*\*Li2.ai \*\*và trên trang [app.li2.ai](http://app.li2.ai):

1. Rút gọn link tại trang web **Li2.ai**
2. Rút gọn link tại trang [app.Li2.ai](http://app.li2.ai)

### Link rút gọn là gì

Link rút gọn giúp biến một URL dài thành một URL ngắn gọn, dễ nhớ và dễ chia sẻ.

- Ví dụ:
  - URL gốc: `https://www.yourwebsite.com/product/abc123`
  - Link ngắn: `https://li2.ai/abc`
  - Link thương hiệu: `https://yourbrand.com/summer-sale`

## Cách rút gọn link tại trang web Li2

Bạn có thể rút gọn link miễn phí ngay trên trang chủ mà không cần đăng nhập tài khoản.

1. Truy cập **li2.ai**
2. Tìm trường **Rút Gọn Link** trên trang chủ.
3. Dán **URL đích**, sau đó nhấp vào **Nhận liên kết miễn phí.**
4. Sao chép link rút gọn đã được tạo.

   <Tip>
     Để sử dụng các tính năng nâng cao như tùy chỉnh URL (đuôi link), tạo mã QR cho link, hoặc xem phân tích chi tiết bạn cần tạo tài khoản và đăng nhập vào [**app.Li2.ai**](http://Li2.ai).
   </Tip>
