## Tùy chỉnh phần đuôi link rút gọn

1. **Đăng nhập** tài khoản tại 👉 [app.li2.ai](http://app.li2.ai).
2. Trong menu bên trái, chọn **Điểm Tiếp Xúc (Touchpoints)**.
3. Tìm link cần chỉnh sửa → nhấp vào biểu tượng **✏️ Edit**.
4. Trong cửa sổ chỉnh sửa, bạn có thể:
   - **Sửa back-half** (ví dụ: `li2.ai/summer-sale`).
   - **Cập nhật URL đích** (nếu muốn redirect sang trang khác).
   - **Đổi tiêu đề** của link.
   - **Thêm hoặc xóa tags**.
5. Kiểm tra **preview link mới** và **version history** hiển thị ngay bên dưới.
6. Nhấp **Lưu thay đổi (Save changes)** để áp dụng.

   <Tip>
     Dùng **tags** để nhóm các link đã chỉnh sửa vào cùng chiến dịch. Điều này giúp bạn dễ dàng phân tích hiệu suất qua từng giai đoạn.
   </Tip>
