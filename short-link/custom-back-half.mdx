## Tùy chỉnh phần đuôi link rút gọn

1. **Đăng nhập** tài khoản tại 👉 [app.li2.ai](http://app.li2.ai).
2. Trong menu bên trái, chọn **Link Rút Gọn**.
3. Tìm link cần chỉnh sửa → nhấp vào ô ba chấm và chọn biểu tượng **✏️ Sửa**.

   ![Tuy Chinh Back Half Link Rut Gon 2 Pn](/images/tuy-chinh-back-half-link-rut-gon-2.png)
4. Tại màn hình chi tiết, nhấp vào biểu tượng **✏️ Edit**
   - **Sửa đuôi link (back-half)** (ví dụ: `li2.ai/6LB9c -> li2.ai/summer-sale`).

   ![Tuy Chinh Back Half Link Rut Gon 1 Pn](/images/tuy-chinh-back-half-link-rut-gon-1.png)
6. Kiểm tra **link rút gọn mới** và lịch sử phiên bản hiển thị ngay bên dưới.
6. Nhấp **Lưu thay đổi (Save changes)** để áp dụng.

   ![Tuy Chinh Back Half Link Rut Gon Pn](/images/tuy-chinh-back-half-link-rut-gon.png)

   <Warning>
     Các đuôi link (back-half) là duy nhất. Bạn không thể tạo hai hoặc nhiều đuôi link trùng tên.
   </Warning>
