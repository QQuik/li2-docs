# Tags Overview

<Info>
**What you'll learn**: How to use tags to organize touchpoints, establish tagging conventions, and leverage tags for filtering and reporting.
</Info>

## What are Tags?

**Tags** are flexible labels you can apply to touchpoints for categorization and filtering. Unlike collections (which group touchpoints into folders), tags allow cross-cutting organization - a touchpoint can have multiple tags.

## Tag Structure

Establish consistent tagging conventions:

**Recommended Format**: `category:value`

**Examples**:
- `campaign:summer-2024`
- `channel:instagram`
- `type:promotional`
- `status:active`
- `region:north-america`

**Benefits of Structured Tags**:
- Easy to understand at a glance
- Prevents tag proliferation
- Enables category-based filtering
- Maintains consistency across team

## Creating and Applying Tags

### During Touchpoint Creation
1. In the creation form, find the "Tags" field
2. Type tag name and press Enter
3. Add multiple tags as needed

### After Creation
1. Navigate to touchpoint detail page
2. Click "+ Add Tag"
3. Select existing tag or create new one

### Bulk Tagging
1. Select multiple touchpoints from list
2. Click **Bulk Actions** → **Add Tags**
3. Choose tags to apply to all selected touchpoints

## Common Tagging Strategies

### Campaign-Based Tags
Track touchpoints by marketing campaign:
- `campaign:black-friday-2024`
- `campaign:product-launch-q2`
- `campaign:brand-awareness`

### Channel-Based Tags
Identify distribution channel:
- `channel:instagram`
- `channel:email`
- `channel:print-qr`
- `channel:in-store`

### Type-Based Tags
Categorize by purpose:
- `type:promotional`
- `type:educational`
- `type:transactional`
- `type:event`

### Status Tags
Track touchpoint lifecycle:
- `status:active`
- `status:paused`
- `status:archived`
- `status:testing`

### Geographic Tags
Organize by location:
- `region:north-america`
- `region:apac`
- `city:new-york`
- `city:singapore`

## Filtering by Tags

**Single Tag Filter**:
1. Click any tag in the tag library
2. View all touchpoints with that tag

**Multi-Tag Filter** (AND logic):
1. Use advanced filter panel
2. Select multiple tags
3. See touchpoints that have ALL selected tags

**Tag Exclusion** (NOT logic):
1. Use advanced filter
2. Select tags to exclude
3. See touchpoints WITHOUT those tags

## Tag Management

### Renaming Tags
1. Go to Tags → Tag Library
2. Click tag → Rename
3. All touchpoints automatically update

### Merging Tags
Combine duplicate or similar tags:
1. Select tags to merge
2. Choose target tag name
3. All touchpoints update to use new tag

### Deleting Tags
<Warning>
**Non-destructive**: Deleting a tag removes it from all touchpoints but doesn't delete the touchpoints themselves.
</Warning>

### Color Coding
Assign colors to tags for visual organization:
1. Click tag → Edit
2. Choose color from palette
3. Tags display with color in lists and filters

## Best Practices

<Tip>
**Start Simple**: Begin with 3-4 tag categories (campaign, channel, type). Add more as needs grow.
</Tip>

<Tip>
**Document Your System**: Create a tagging guide for your team documenting approved tag categories and naming conventions.
</Tip>

<Tip>
**Review Regularly**: Monthly audit to merge duplicate tags, archive old campaign tags, and maintain consistency.
</Tip>

<Warning>
**Avoid Tag Explosion**: Without conventions, tag counts grow unmanageable. Enforce naming standards early.
</Warning>

## Tags vs Collections

| Feature | Tags | Collections |
|---------|------|-------------|
| **Structure** | Flat labels | Hierarchical folders |
| **Quantity per Touchpoint** | Unlimited | One collection |
| **Organization** | Multi-dimensional | Single-dimension |
| **Analytics** | Filter-based | Aggregated dashboard |
| **Best For** | Cross-cutting categories | Campaign/client grouping |

**Use Both**: Combine tags and collections for powerful organization. Example: "Summer Campaign" collection with tags for `channel:`, `type:`, `region:`
