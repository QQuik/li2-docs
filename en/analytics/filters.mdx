# Filters & Date Ranges

<Info>
**What you'll learn**: How to filter analytics data by various criteria and select appropriate date ranges for meaningful insights.
</Info>

## Available Filters

### Tag Filters
Filter analytics to show only touchpoints with specific tags:
- Single tag: `campaign:summer-2024`
- Multiple tags (AND): Show touchpoints with ALL selected tags
- Tag exclusion (NOT): Exclude touchpoints with certain tags

### Collection Filters
View analytics for entire collections:
- Select one or more collections
- See aggregated metrics for all touchpoints in those collections

### Touchpoint Filters
Narrow down to specific touchpoints:
- Search by touchpoint name or slug
- Select from dropdown list
- Compare multiple touchpoints side-by-side

### Device Type Filters
Segment by device category:
- Mobile
- Desktop
- Tablet

### Geographic Filters
Filter by location:
- Country
- Region/State
- City

## Date Range Selection

### Quick Ranges
Predefined date ranges for common use cases:
- Today
- Yesterday
- Last 7 days
- Last 30 days
- Last 90 days
- This month
- Last month
- This year

### Custom Date Range
Select specific start and end dates:
1. Click date range selector
2. Choose start date
3. Choose end date
4. Apply filter

### Date Comparison
Compare periods side-by-side:
- Current period vs previous period
- Year-over-year comparison
- Custom period comparison

## Combining Filters

Apply multiple filters simultaneously for granular insights:

**Example 1: Campaign Performance by Channel**
- Filter: `campaign:summer-2024`
- Filter: `channel:instagram`
- Date Range: Last 30 days
- Result: Instagram performance for summer campaign

**Example 2: Mobile Traffic by Region**
- Filter: Device = Mobile
- Filter: Country = United States
- Date Range: This month
- Result: US mobile traffic trends

## Saving Filter Presets

Save frequently used filter combinations:
1. Apply desired filters
2. Click "Save as Preset"
3. Name your preset (e.g., "Summer Campaign - Social Media")
4. Access anytime from Presets dropdown

## Best Practices

<Tip>
**Compare Apples to Apples**: When comparing periods, use same day-of-week ranges (Mon-Sun) for accurate comparison.
</Tip>

<Tip>
**Baseline with 30 Days**: For most campaigns, 30-day ranges provide statistical significance while remaining actionable.
</Tip>
