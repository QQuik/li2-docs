# Analytics Overview

<Info>
**What you'll learn**: Navigate Li2.ai analytics, understand key metrics, and extract actionable insights from touchpoint performance data.
</Info>

## Core Metrics

### Total Clicks & Scans
- Combined count of all interactions
- Short link clicks + QR code scans + NFC taps
- Real-time updates

### Unique Visitors
- Distinct users who clicked/scanned
- Deduplicated by device fingerprint
- Useful for reach measurement

### Click-Through Rate (CTR)
- Clicks / Impressions (when impression tracking enabled)
- Industry benchmark comparison

### Conversion Rate
- Conversions / Total Clicks
- Requires conversion goal setup
- Track ROI and goal completions

## Analytics Categories

### Geographic Analytics
**Country-Level Data**
- Traffic distribution by country
- Interactive world map visualization
- Top countries ranking

**City-Level Data**
- Granular location insights
- Identify local markets
- Regional campaign optimization

### Device Analytics
**Device Type**
- Mobile, Desktop, Tablet breakdown
- Percentage distribution
- Trend over time

**Operating System**
- iOS, Android, Windows, macOS, Linux
- Version distribution

**Browser**
- Chrome, Safari, Firefox, Edge, etc.
- Mobile vs desktop browsers

### Referrer Analytics
**Traffic Sources**
- Direct scans (QR codes, NFC)
- Social media platforms
- Websites
- Email clients
- Unknown

**UTM Campaign Tracking**
- Source, Medium, Campaign breakdown
- Compare channel performance
- ROI by marketing channel

### Time-Series Analytics
**Traffic Patterns**
- Hourly, daily, weekly, monthly views
- Peak traffic identification
- Seasonal trend analysis

**Custom Date Ranges**
- Compare specific periods
- Year-over-year analysis
- Campaign duration insights

## Analytics Dashboard

### Organization Dashboard
View aggregated metrics across all touchpoints:
- Total organization performance
- Top performing touchpoints
- Geographic heatmap
- Device distribution
- Traffic trends

### Touchpoint Dashboard
Individual touchpoint analytics:
- Click/scan timeline
- Geographic breakdown
- Device and browser stats
- Referrer sources
- Recent activity log

### Collection Dashboard
Aggregated analytics for collections:
- Collection-level metrics
- Compare touchpoints within collection
- Campaign performance overview

## Real-Time Data

Li2.ai updates analytics in real-time:
- See clicks/scans within seconds
- Monitor live campaign performance
- Immediate feedback on changes
- No batch processing delays

## Data Export

Export analytics for external analysis:

**CSV Export**
- Raw click data with timestamps
- Geographic, device, referrer details
- Import into Excel, Google Sheets
- Custom reporting and visualization

**PDF Reports**
- Formatted analytics reports
- Charts and visualizations included
- Share with stakeholders
- Schedule automated reports (coming soon)

## Privacy & Compliance

Li2.ai analytics respect user privacy:
- No personal data collection
- Device fingerprinting (not cookies)
- IP anonymization
- GDPR compliant
- No cross-site tracking

## Best Practices

<Tip>
**Set Baselines**: Track 7-30 day averages to understand normal performance before optimizing.
</Tip>

<Tip>
**Segment Analysis**: Don't just look at totals - segment by device, geography, source for deeper insights.
</Tip>

<Tip>
**Regular Reviews**: Check analytics weekly for active campaigns, monthly for long-term touchpoints.
</Tip>
