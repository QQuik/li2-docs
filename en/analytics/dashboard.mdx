# Organization Dashboard

<Info>
**What you'll learn**: Navigate the organization analytics dashboard to monitor overall performance and identify trends across all touchpoints.
</Info>

## Dashboard Overview

The Organization Dashboard provides bird's-eye view of all touchpoint performance:

### Key Metrics Panel
**Total Statistics**
- Total Clicks/Scans: All-time aggregate
- This Month: Current month performance
- Growth Rate: Month-over-month change
- Active Touchpoints: Total touchpoint count

### Top Performers
**Best Touchpoints** (Last 30 Days)
- Ranked by total interactions
- Click/scan counts
- Quick access to touchpoint details
- Identify successful campaigns

**Trending Up**
- Fastest growing touchpoints
- Percentage growth indicators
- New touchpoint highlights

**Needs Attention**
- Declining performance alerts
- Zero-click touchpoints
- Error/broken link notifications

## Visualization Widgets

### Traffic Timeline
Interactive time-series chart:
- Hourly, daily, weekly, monthly views
- Zoom and pan functionality
- Compare multiple date ranges
- Export chart as image

### Geographic Heatmap
World map visualization:
- Color-coded by traffic volume
- Click countries for city-level data
- Identify expansion opportunities
- Regional performance comparison

### Device Distribution
Pie/donut chart showing:
- Mobile vs Desktop vs Tablet
- Percentage breakdown
- Trend indicators
- Platform-specific insights

### Channel Performance
Bar chart comparing:
- Direct (QR scans, NFC)
- Social media
- Email
- Website referrals
- UTM campaign sources

## Filtering Dashboard Data

Apply filters to focus analysis:
- **Date Range**: Last 7/30/90 days, custom ranges
- **Tags**: Filter by campaign, channel, type tags
- **Collections**: View specific collection performance
- **Device Type**: Mobile-only or desktop-only views

## Dashboard Customization

Personalize your dashboard:
- **Widget Arrangement**: Drag to reorder
- **Show/Hide Widgets**: Toggle widgets on/off
- **Default Date Range**: Set preferred time period
- **Dashboard Themes**: Light or dark mode

## Scheduled Reports

Configure automated reports:
- **Frequency**: Daily, weekly, monthly
- **Recipients**: Team email addresses
- **Content**: Selected metrics and charts
- **Format**: PDF or CSV

## Best Practices

<Tip>
**Weekly Review Routine**: Every Monday, review last week's performance to spot trends early.
</Tip>

<Tip>
**Set Alerts**: Configure notifications for milestone achievements or performance drops.
</Tip>
