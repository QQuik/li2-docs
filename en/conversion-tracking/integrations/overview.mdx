<Info>
Integrate Li2 conversion tracking with your favorite tools and platforms without writing code directly on your website.

Third-party integrations allow you to implement Li2 conversion tracking through tools you already use, providing:

  <Card title="No Code Changes" icon="code-simple">

  </Card>

  <Card title="Version Control" icon="clock-rotate-left">
    Test changes in preview mode before publishing
  </Card>

  <Card title="Easy Updates" icon="arrows-rotate">

## Choosing the Right Integration

<AccordionGroup>
  <Accordion title="I want to track conversions without touching code">
    **Recommended**: [Google Tag Manager](./google-tag-manager)

    GTM is perfect if you:
    - Don't have developer access
    - Want to control tracking yourself
    - Need to test before deploying
    - Already use GTM for other tools
  </Accordion>

  <Accordion title="I run an online store (Shopify, WooCommerce)">
    **Coming Soon**: E-commerce platform integrations

    Native integrations will provide:
    - Automatic order tracking
    - No manual setup required
    - Built-in e-commerce features
    - One-click installation

    **Current Solution**: Use [Google Tag Manager](./google-tag-manager) or [Direct SDK Installation](../setup)
  </Accordion>

  <Accordion title="I need to connect Li2 with my CRM/Marketing tools">
    **Coming Soon**: Zapier and Segment integrations

    These will enable:
    - Automatic data syncing
    - Multi-platform workflows
    - No-code automation
    - Real-time updates

    **Current Solution**: Use webhooks or [API integration](../events) for server-side tracking
  </Accordion>

  <Accordion title="I have developers and want full control">
    **Recommended**: [Direct SDK Installation](../setup)

    Direct integration provides:
    - Maximum flexibility
    - Custom event tracking
    - Server-side tracking option
    - Full API access

    You can also combine direct SDK with GTM for maximum control.
  </Accordion>
</AccordionGroup>
