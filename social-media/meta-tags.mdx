<Note>
  **Premium Feature**: Custom meta tags are available for Premium tier users. Without premium access, touchpoints use default Li2.ai meta tags when shared.
</Note>

**Các bước thực hiện**:

1. Đăng nhập vào tài khoản Li2 tại [app.li2.ai](http://app.li2.ai).
2. Trong menu bên trái, chọn **Link Rút Gọn**.
3. Tìm link cần chỉnh sửa  → nhấp vào ô ba chấm và chọn biểu tượng **✏️ Sửa**. 
4. Chọn tab **Nâng cao**. Bật công tắc **Meta Tags**.
4. Tùy chỉnh các thông tin:
   - **Hình ảnh**: Tải lên hình ảnh đại diện cho link.
   - **Tiêu đề**: Nhập tiêu đề muốn hiển thị.
   - **Mô tả**: Viết mô tả ngắn gọn về nội dung link.
5. Xem trước giao diện hiển thị ở khung bên phải.
6. Nhấn **Lưu thay đổi** để áp dụng.
