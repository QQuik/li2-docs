## Accessing UTM Parameters

### UTM Builder Location

<Steps>
  <Step title="Navigate to touchpoint advanced settings">
    1. Go to **Touchpoints** and select your touchpoint
    2. Click on the **Advanced** tab
    3. Find the **UTM Parameters** section
    4. Toggle **UTM Parameters** to **ON** to enable UTM tracking
  </Step>
  <Step title="Understanding UTM builder interface">
    The UTM Parameters section includes:

    - **Required Fields**: Source (mandatory for all campaigns)
    - **Optional Fields**: Campaign, Medium, Term, Content
    - **Analytics Integration**: GA4 ID/GTM ID for direct tracking
    - **Redirect Example**: Preview of final URL with UTM parameters

      ![UTM Parameter Management Pn](/images/UTM-parameter-management.png)
  </Step>
</Steps>

## UTM Parameter Configuration

### Required Parameters

<AccordionGroup>
  <Accordion title="Source (Required)">
    **Purpose**: Identifies where traffic originates from

    **Built-in Suggestions**:

    - **google** - Google Ads, Google search campaigns
    - **facebook** - Facebook and Meta advertising
    - **instagram** - Instagram-specific campaigns
    - **linkedin** - LinkedIn advertising and content
    - **twitter** - Twitter/X campaigns and posts
    - **email** - Email marketing campaigns
    - **newsletter** - Newsletter-specific tracking
    - **bing** - Microsoft Bing advertising

    **Custom Sources**: Enter any custom source name for specific platforms or campaigns
  </Accordion>
</AccordionGroup>

### Optional Parameters

<AccordionGroup>
  <Accordion title="Campaign">
    **Purpose**: Identifies specific marketing campaigns

    **Example Values**:

    - **holiday_special** - Seasonal promotional campaigns
    - **birthday_promotion** - Special event campaigns
    - **product_launch** - New product introduction campaigns
    - **summer_2025** - Time-based campaign identification

    **Best Practices**:

    - Use consistent naming conventions across campaigns
    - Include dates or periods for time-sensitive campaigns
    - Keep names descriptive but concise
  </Accordion>
  <Accordion title="Medium">
    **Purpose**: Categorizes the marketing medium or channel type

    **Built-in Suggestions**:

    - **cpc** - Cost-per-click advertising
    - **email** - Email marketing campaigns
    - **social** - Social media posts and campaigns
    - **display** - Display advertising and banners
    - **referral** - Referral program tracking
    - **organic** - Organic social media posts
    - **paid-social** - Paid social media advertising
    - **affiliate** - Affiliate marketing programs

    **Strategic Value**: Groups similar marketing activities for performance comparison
  </Accordion>
  <Accordion title="Term">
    **Purpose**: Tracks specific keywords or targeting criteria

    **Example Usage**:

    - **iphone_15** - Product-specific keywords
    - **wireless_earbuds** - Category targeting
    - **hotel_deal** - Service-specific terms
    - **marketing_automation** - B2B keyword targeting

    **Applications**: Particularly useful for paid search campaigns and keyword tracking
  </Accordion>
  <Accordion title="Content">
    **Purpose**: Differentiates between similar content or creative variations

    **Example Values**:

    - **video_ad_short** - Video advertisement variations
    - **popup_offer_button** - Specific call-to-action elements
    - **banner_ad_1** - Creative testing and optimization
    - **header_cta** - Placement-specific tracking

    **A/B Testing**: Essential for comparing different creative approaches and placements
  </Accordion>
</AccordionGroup>

## Analytics Integration

### Google Analytics 4 (GA4) Integration

<Steps>
  <Step title="Configure GA4 tracking">
    1. In the **UTM Parameters** section, find **GA4 ID / GTM ID** field
    2. Enter your Google Analytics 4 measurement ID (format: GTM-XXXXXXX or G-XXXXXXX)
    3. UTM parameters will be sent directly to your GA4 property
    4. Works even if destination pages don't have GA4 tracking code
  </Step>
  <Step title="Benefits of direct GA4 integration">
    **Enhanced Tracking Capabilities**:

    - **Bypasses Missing Tracking**: Send data even without destination page analytics
    - **Consistent Attribution**: Reliable campaign tracking across all touchpoints
    - **Immediate Data Flow**: UTM parameters reach GA4 regardless of destination setup
    - **Complete Campaign View**: Track full customer journey from Li2.ai to conversion
  </Step>
</Steps>

### Google Tag Manager (GTM) Integration

<Note>
  **GTM Support**: Enter your Google Tag Manager container ID (GTM-XXXXXXX format) to send UTM parameter data directly to your GTM setup for advanced tracking configurations.
</Note>

**GTM Integration Benefits**:

- **Advanced Event Tracking**: Trigger custom events based on UTM parameters
- **Cross-Platform Attribution**: Connect Li2.ai data with other marketing platforms
- **Custom Dimension Tracking**: Send UTM data to custom GA4 dimensions
- **Enhanced E-commerce Tracking**: Attribute transactions to specific campaigns

## UTM Parameter Management

### URL Construction and Preview

<Steps>
  <Step title="Automatic URL building">
    **How UTM URLs Are Created**:

    1. Li2.ai automatically appends UTM parameters to your destination URL
    2. **Redirect Example** shows the final URL users will visit
    3. Parameters are properly formatted and encoded
    4. No manual URL construction required
  </Step>
  <Step title="UTM overwrite settings">
    **Allow UTM Overwrite Option**:

    - **When Enabled**: UTM parameters in short URLs override stored ones
    - **When Disabled**: Stored UTM parameters always take precedence
    - **Use Case**: Allows dynamic UTM modification for different campaigns using the same touchpoint
  </Step>
</Steps>

### Campaign Tracking Strategy

<AccordionGroup>
  <Accordion title="Consistent naming conventions">
    **Professional UTM Standards**:

    - **Source**: Use lowercase, standardized platform names
    - **Medium**: Stick to common medium types (cpc, email, social, display)
    - **Campaign**: Use descriptive, date-inclusive names
    - **Term/Content**: Include specific identifiers for A/B testing

    **Example UTM Structure**:

    ```
    Source: facebook
    Medium: paid-social
    Campaign: summer_2025_sale
    Content: video_ad_v2
    ```
  </Accordion>
  <Accordion title="Multi-channel campaign coordination">
    **Campaign Organization**:

    - **Consistent Campaign Names**: Use same campaign parameter across all channels
    - **Channel-Specific Sources**: Different sources for each platform (facebook, google, email)
    - **Medium Differentiation**: Distinguish between organic and paid efforts
    - **Content Variations**: Track different creative approaches within campaigns

    **Strategic Benefits**:

    - Compare performance across different marketing channels
    - Understand which creative variations work best
    - Track customer journey across multiple touchpoints
    - Optimize marketing spend based on UTM attribution data
  </Accordion>
</AccordionGroup>

## Troubleshooting UTM Parameters

<AccordionGroup>
  <Accordion title="UTM data not appearing in analytics">
    **Common Issues and Solutions**:

    **Missing GA4/GTM ID**:

    - Verify GA4 ID or GTM ID is correctly entered
    - Ensure ID format matches GTM-XXXXXXX or G-XXXXXXX
    - Check that GA4 property is active and receiving data

    **URL Construction Problems**:

    - Review Redirect Example to verify UTM parameters are properly appended
    - Ensure destination URL doesn't already contain UTM parameters
    - Check Allow UTM Overwrite settings if using dynamic UTMs
  </Accordion>
  <Accordion title="UTM parameter formatting">
    **Best Practice Formatting**:

    - **Use Lowercase**: Avoid mixed case in UTM values
    - **No Spaces**: Use underscores or hyphens instead of spaces
    - **Special Characters**: Avoid special characters that might break URLs
    - **Length Limits**: Keep UTM values concise for better URL management

    **Validation Tips**:

    - Use Li2.ai's built-in suggestions when possible
    - Test URLs in analytics platforms before large campaigns
    - Verify UTM data appears correctly in GA4 reports
  </Accordion>
</AccordionGroup>
