# Analytics & Tracking Overview

Monitor your touchpoint performance with Li2.ai's comprehensive analytics system. Track visits, analyze traffic sources, understand your audience demographics, and optimize your marketing campaigns with data-driven insights.

## Analytics Access Points

<CardGroup cols={2}>
  <Card title="Global Analytics Dashboard" icon="chart-line" color="#7C3AED">
    **Main Analytics Section**: View performance across all touchpoints with filtering and comparison capabilities.
  </Card>
  <Card title="Individual Touchpoint Analytics" icon="magnifying-glass" color="#A855F7">
    **Touchpoint Detail View**: Analyze specific touchpoint performance in the Analytics tab of touchpoint details.
  </Card>
</CardGroup>

**Two Ways to Access Analytics**:
- **Global View**: Navigate to **Analytics** in the main sidebar for account-wide insights
- **Touchpoint-Specific**: Open any touchpoint and click the **Analytics** tab for detailed performance data

---

## Core Analytics Features

### Key Performance Metrics

<Steps>
  <Step title="Overall Visits Tracking">
    **Primary Metrics Available**:
    - **Total Visits**: Complete visitor count across all channels
    - **Click**: Direct link clicks through LINK channel
    - **Scan**: QR code scans through QR channel
    - **Top Performance**: Highest performing touchpoint with daily metrics
  </Step>
  
  <Step title="Multi-channel insights">
    **Channel Data Available**:
    - Track combined performance across LINK and QR channels
    - Monitor overall touchpoint effectiveness
    - Identify daily performance trends
    - View total engagement metrics
  </Step>
</Steps>

### Traffic Source Analysis

<AccordionGroup>
  <Accordion title="By Referrer tracking">
    **Traffic Source Breakdown**:
    - **Direct**: Users accessing touchpoints directly
    - **Social Media Platforms**: Facebook, Twitter, LinkedIn, Instagram, and other social networks
    - **Messaging Apps**: WhatsApp, Telegram, and other messaging platforms
    - **Other Applications**: Detectable apps and platforms when possible
    - **Unknown**: Unidentified traffic sources
    
    **Visual Representation**:
    - Bar chart showing relative volume from each referrer
    - Easy identification of top-performing traffic sources
    - Track referrer effectiveness across touchpoints
  </Accordion>
  
  <Accordion title="Device and OS analytics">
    **Device Insights**:
    - **Operating System**: Mac OS X, iOS, Android, Windows breakdown
    - **Device Type**: Desktop vs mobile usage patterns
    - **Visual Pie Chart**: Proportional device usage display
    - **Visitor Count**: Total visitors from each device type
    
    **Strategic Value**:
    - Optimize touchpoint experience for primary device types
    - Understand audience technology preferences
    - Plan mobile-first or desktop-focused campaigns
  </Accordion>
</AccordionGroup>

### Geographic Analytics

<CardGroup cols={2}>
  <Card title="Global Geographic Tracking" icon="globe" color="#10B981">
    **Worldwide Coverage**: Track visits from any country or region where your touchpoints are accessed.
  </Card>
  <Card title="Visit Rate Analysis" icon="percentage" color="#10B981">
    **Performance Metrics**: Monitor visit rates and conversion percentages by geographic location for regional insights.
  </Card>
</CardGroup>

**Geographic Data Includes**:
- **Country Code**: ISO country identification for all countries
- **Visit Count**: Number of visits from each location
- **Visit Rate**: Percentage conversion rates by region
- **Visual Map**: Geographic distribution visualization

## Filtering and Analysis Tools

### Available Filter Options

<Steps>
  <Step title="Tag-based filtering">
    1. Click **"Filter by Tag"** dropdown in the analytics interface
    2. Select specific tags to analyze performance of tagged touchpoints
    3. View analytics for specific campaigns, channels, or audience segments
    4. Compare performance across different tag categories
  </Step>
  
  <Step title="Date range analysis">
    1. Use the **date picker** to select custom time periods
    2. Analyze performance trends over specific periods
    3. Compare different time ranges for optimization insights
    4. Track campaign performance during specific periods
  </Step>
</Steps>

## Analytics Dashboard Components

### Global Analytics View

**Main Dashboard Elements**:
- **Overall Visits Summary**: Total visits with click/scan breakdown
- **Top Performance**: Highest performing touchpoint identification
- **Referrer Analysis**: Bar chart showing traffic source distribution
- **Device Breakdown**: Pie chart of OS and device type usage
- **Geographic Distribution**: Location-based visit analysis with visit rates

### Individual Touchpoint Analytics

**Touchpoint-Specific Insights**:
- **Focused Performance Data**: Analytics for single touchpoint
- **Same Metric Categories**: Visits, referrers, devices, and location data
- **Comparative Context**: Performance relative to account averages
- **Detailed Attribution**: Specific traffic source analysis for optimization

<Note>
**Access Method**: Navigate to any touchpoint details page and click the **Analytics** tab to view touchpoint-specific performance data with the same comprehensive metrics available in the global view.
</Note>
