## Current Integration: Parcelvoy

### Multi-Channel Marketing Automation

<Note>
**Ready to Use**: Parcelvoy integration enables automated messaging and customer engagement workflows through touchpoint interactions.
</Note>

**Key Capabilities**:
- **Automated Messaging**: Trigger email, SMS, and push notifications based on touchpoint interactions
- **Customer Engagement**: Create personalized customer journeys from touchpoint data
- **Multi-Channel Campaigns**: Coordinate messaging across email, SMS, and other channels
- **Growth Analytics**: Track customer engagement and campaign performance

<Steps>
  <Step title="Connect Parcelvoy">
    1. Navigate to **Integration** in the main sidebar
    2. Click on the **Parcelvoy** integration card
    3. Follow the connection setup process
    4. Configure your automated messaging workflows
  </Step>
  
  <Step title="Set up automation">
    - Create automated email sequences from touchpoint interactions
    - Set up SMS campaigns triggered by QR code scans
    - Configure customer engagement workflows
    - Monitor performance through integrated analytics
  </Step>
</Steps>
