## Configuring Journey Entrances

### Accessing Journey Settings

<Steps>
  <Step title="Navigate to touchpoint advanced settings">
    1. Go to **Touchpoints** and select your desired touchpoint
    2. Click on the **Advanced** tab in touchpoint details
    3. Find the **Parcelvoy** section
    4. Ensure Parcelvoy integration is enabled (toggle ON)
  </Step>
  
  <Step title="Configure journey entrance">
    **Journey Configuration Fields**:
    - **Integration**: Select your configured Parcelvoy integration
    - **Journey**: Choose the specific journey from dropdown menu
    - **Entrance**: Select the entrance point within the chosen journey
    
    **Setup Process**:
    1. Click **"Select integration"** dropdown
    2. Choose your active Parcelvoy integration
    3. Select the appropriate journey from **"Select journey"** dropdown
    4. Choose the entrance point from **"Select entrance"** options
  </Step>
  
  <Step title="Save configuration">
    1. Review your journey entrance configuration
    2. Click **"Save Changes"** to activate the automation
    3. Journey triggers will now activate when users interact with the touchpoint
    4. Monitor journey performance through Parcelvoy analytics
  </Step>
</Steps>

### Journey Configuration Options

<AccordionGroup>
  <Accordion title="Integration selection">
    **Choosing the Right Integration**:
    - **Multiple Integrations**: Select from configured Parcelvoy integrations
    - **Integration Status**: Only active integrations appear in dropdown
    - **Account Mapping**: Each integration corresponds to specific Parcelvoy account
    - **Permissions**: Ensure integration has access to desired journeys
  </Accordion>
  
  <Accordion title="Journey and entrance mapping">
    **Journey Selection Process**:
    - **Available Journeys**: Dropdown shows all active journeys from selected integration
    - **Journey Status**: Only published/active journeys are available for selection
    - **Entrance Points**: Choose specific entry points within multi-step journeys
    - **Dynamic Updates**: Journey list updates automatically based on Parcelvoy changes
    
    **Entrance Configuration**:
    - **Entry Triggers**: Define when users enter the journey
    - **Conditional Logic**: Set up rules for journey entrance qualification
    - **User Segmentation**: Target specific user segments through entrance criteria
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Configuration issues">
    **Common Setup Problems**:
    - **Missing Integrations**: Verify Parcelvoy integration is properly configured and active
    - **Empty Dropdowns**: Ensure Parcelvoy account has published journeys available
    - **Permission Errors**: Check API key permissions for journey access
    - **Premium Access**: Confirm Premium tier subscription for advanced features
    
    **Resolution Steps**:
    - Refresh integration connection in settings
    - Verify journey status in Parcelvoy platform
    - Test API key permissions and regenerate if necessary
    - Contact support for Premium tier verification
  </Accordion>
  
  <Accordion title="Journey trigger issues">
    **Automation Problems**:
    - **Journeys Not Triggering**: Verify touchpoint interactions are generating events
    - **Duplicate Entrances**: Check for multiple journey configurations on same touchpoint
    - **Wrong Journey Execution**: Confirm correct journey and entrance selection
    - **Delayed Triggers**: Monitor integration latency and event processing times
    
    **Debugging Process**:
    - Test touchpoint interactions and monitor journey logs
    - Verify journey entrance criteria in Parcelvoy platform
    - Check integration event logs for error messages
    - Ensure journey is active and properly configured in Parcelvoy
  </Accordion>
</AccordionGroup>
