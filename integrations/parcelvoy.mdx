## Setting Up Parcelvoy Integration

### Step 1: Get Your Parcelvoy API Key

<Steps>
  <Step title="Access Parcelvoy">
    **Using Li2.ai Hosted Service**:
    1. Visit [engage.li2.ai](https://engage.li2.ai)
    2. Create your Parcelvoy account or log in to existing account
    3. Set up your marketing automation workspace
    
    **Using Self-Hosted Parcelvoy**:
    - Access your own Parcelvoy installation
    - Navigate to your project settings
  </Step>
  
  <Step title="Generate API key">
    1. Navigate to your **Project Settings** in Parcelvoy
    2. Find the **API Keys** or **Integrations** section
    3. Generate a new **Secret API Key** for Proxify integration
    4. Copy the API key for use in Proxify setup
    5. Keep this key secure as it provides access to your Parcelvoy data
  </Step>
</Steps>

### Step 2: Configure Integration in Proxify

<Steps>
  <Step title="Access Parcelvoy integration">
    1. Navigate to **Integration** in the Proxify sidebar
    2. Find the **Parcelvoy** integration card
    3. Click on the Parcelvoy integration to access configuration
    4. You'll see the integration overview and setup options
  </Step>
  
  <Step title="Create integration connection">
    1. Click **"Create Integration"** button
    2. The integration configuration dialog opens
    3. Fill in the required fields:
       - **Name**: Descriptive name for your integration (e.g., "Main Parcelvoy")
       - **API Key**: Paste your Secret API Key from Parcelvoy
    4. Click **"Save"** to establish the connection
  </Step>
  
  <Step title="Verify connection">
    1. Proxify will test the API connection automatically
    2. Successful connection will show in your integrations list
    3. You can now use Parcelvoy with your touchpoint campaigns
    4. The integration will appear under "All Integrations" section
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Connection issues">
    **API Key Problems**:
    - Verify API key is copied correctly without extra spaces
    - Ensure API key has proper permissions in Parcelvoy project
    - Check that Parcelvoy project is active and accessible
    - Regenerate API key if connection continues to fail
    
    **Integration Status**:
    - Refresh integration page to check connection status
    - Contact support if integration shows as disconnected
    - Verify network connectivity between Proxify and Parcelvoy
  </Accordion>
  
  <Accordion title="Data flow issues">
    **Missing Data in Parcelvoy**:
    - Verify integration is active and properly configured
    - Check that touchpoints are generating interactions
    - Ensure Parcelvoy workspace is set up to receive data
    - Monitor integration logs for error messages
    
    **Automation Not Triggering**:
    - Verify automation rules are properly configured in Parcelvoy
    - Check that trigger conditions match touchpoint data
    - Test with sample touchpoint interactions
    - Review Parcelvoy automation logs for debugging
  </Accordion>
</AccordionGroup>
