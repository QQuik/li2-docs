## Tải mã QR nhanh tại trang web [Li2](http://Li2.ai)

Sau khi tạo thành công mã QR ngay tại trang web Li2, hệ thống sẽ tạo ra mã QR hiển thị ở khung bên phải. Bạn có thể:

- Nhấn **Download QR Code** để tải file mã QR.
- Hoặc chọn **Copy QR Image** để sao chép trực tiếp hình ảnh QR.

## Download Format Options

### PNG Formats

Choose from multiple PNG sizes based on your intended use:

<AccordionGroup>
  <Accordion title="512 x 512 pixels (Recommended)">
    **Best for**:

    - Digital displays and websites
    - Email signatures and newsletters
    - Social media posts
    - Mobile applications
    - Quick print materials

    **Benefits**:

    - Smaller file size for fast loading
    - Perfect for most digital applications
    - Good quality for standard print sizes
    - Recommended starting point for most users
  </Accordion>
  <Accordion title="1024 x 1024 pixels (High Quality)">
    **Best for**:

    - Business cards and brochures
    - Marketing flyers and handouts
    - Product packaging and labels
    - Professional print materials
    - Medium-format displays

    **Benefits**:

    - Higher resolution for crisp print quality
    - Better detail retention when resized
    - Professional appearance in print materials
    - Good balance of quality and file size
  </Accordion>
  <Accordion title="2048 x 2048 pixels (Maximum Quality)">
    **Best for**:

    - Large format printing (posters, banners)
    - Trade show displays and signage
    - Professional design workflows
    - Future-proofing for unknown applications
    - High-end marketing materials

    **Benefits**:

    - Maximum resolution available
    - Excellent for large-scale printing
    - Professional quality for any application
    - Can be resized down without quality loss
  </Accordion>
</AccordionGroup>

## Choosing the Right Format

### Quick Selection Guide

<Steps>
  <Step title="Determine your primary use">
    **Digital Use**: Start with **512x512 PNG** **Print Materials**: Choose **1024x1024 PNG** **Large Format**: Use **2048x2048 PNG** or **SVG** **Professional Design**: Choose **SVG** format
  </Step>
  <Step title="Consider your specific application">
    **Business Cards**: 512x512 or 1024x1024 PNG **Websites/Email**: 512x512 PNG **Brochures/Flyers**: 1024x1024 PNG **Posters/Banners**: 2048x2048 PNG or SVG **Professional Printing**: SVG preferred
  </Step>
  <Step title="Download and test">
    1. Download your chosen format
    2. Test in your intended application
    3. Verify quality and scanning functionality
    4. Download additional formats if needed
  </Step>
</Steps>

### Application-Specific Recommendations

<AccordionGroup>
  <Accordion title="Digital marketing">
    **Email Signatures**: 512x512 PNG

    - Fast loading in email clients
    - Clear at typical email display sizes
    - Small file size for email attachments

    **Social Media**: 512x512 PNG

    - Quick loading on mobile devices
    - Appropriate for social media post sizes
    - Good quality for profile sharing

    **Websites**: 512x512 PNG or SVG

    - PNG for standard web use
    - SVG for responsive design needs
  </Accordion>
  <Accordion title="Print marketing">
    **Business Cards**: 1024x1024 PNG

    - High enough resolution for crisp printing
    - Good detail retention at small sizes
    - Professional appearance

    **Brochures and Flyers**: 1024x1024 PNG

    - Excellent print quality
    - Clear scanning from printed materials
    - Professional marketing appearance

    **Large Format**: SVG or 2048x2048 PNG

    - SVG preferred for infinite scalability
    - PNG for maximum resolution when SVG not supported
  </Accordion>
</AccordionGroup>

## Download Process

### Step-by-Step Download

<Steps>
  <Step title="Access download options">
    Use either access method:

    - **Quick**: Three dots menu from touchpoint list
    - **Full**: Download button from touchpoint details
  </Step>
  <Step title="Select format and size">
    1. Click on your preferred format option
    2. **PNG**: Choose from 512x512, 1024x1024, or 2048x2048
    3. **SVG**: Vector format option
    4. Download begins immediately
  </Step>
  <Step title="Save and organize">
    1. Choose appropriate filename for organization
    2. Save to easily accessible location
    3. Consider creating folders by campaign or application
    4. Keep original files for future use
  </Step>
</Steps>

## Quality Assurance

### Testing Downloaded QR Codes

<Steps>
  <Step title="Verify file quality">
    1. Open downloaded file in appropriate software
    2. Check image clarity and sharpness
    3. Verify colors display correctly
    4. Ensure logo (if used) is clear and centered
  </Step>
  <Step title="Test scanning functionality">
    1. Display QR code at intended size
    2. Test scanning with multiple devices
    3. Verify redirect works correctly
    4. Try different lighting conditions
  </Step>
  <Step title="Application testing">
    1. Test in final application (print, web, etc.)
    2. Verify appearance meets quality standards
    3. Check scanning from actual implementation
    4. Make adjustments if needed
  </Step>
</Steps>

### Common Quality Checks

<AccordionGroup>
  <Accordion title="Print quality verification">
    **Before Large Print Runs**:

    - Print test copy at actual size
    - Verify QR code scans from printed version
    - Check color accuracy and sharpness
    - Test with intended printing process and materials

    **Quality Standards**:

    - QR code should be crisp and clear
    - Colors should match your brand guidelines
    - Logo (if used) should be visible and centered
    - Scanning should work reliably from print
  </Accordion>
  <Accordion title="Digital display verification">
    **Screen Display Testing**:

    - Test on various device types and screen sizes
    - Verify colors display correctly on different monitors
    - Check scanning from screen displays
    - Ensure fast loading and appropriate file size

    **Compatibility Checks**:

    - Test in email clients (if using in signatures)
    - Verify social media platform compatibility
    - Check website integration and responsive behavior
    - Ensure mobile device scanning works reliably
  </Accordion>
</AccordionGroup>

## Troubleshooting Downloads

<AccordionGroup>
  <Accordion title="Download not available">
    **Check QR channel status**:

    - Verify QR channel is enabled in touchpoint
    - Confirm touchpoint is active
    - Ensure QR code has been generated successfully

    **Access method verification**:

    - QR-enabled touchpoints show QR icon in list
    - Download option appears in three dots menu
    - Download button visible in touchpoint details QR section
  </Accordion>
  <Accordion title="File quality issues">
    **Poor quality or pixelation**:

    - Try higher resolution PNG format
    - Use SVG for scalable quality
    - Verify file opened in appropriate software

    **Color display problems**:

    - Check monitor calibration
    - Verify color settings in design software
    - Test on multiple devices for consistency
  </Accordion>
  <Accordion title="File format compatibility">
    **SVG not supported**:

    - Use PNG format for basic applications
    - Convert SVG to PNG using design software if needed
    - Choose highest resolution PNG for quality retention

    **File size concerns**:

    - Use 512x512 PNG for smaller file sizes
    - SVG often smaller than high-resolution PNG
    - Compress PNG files if needed for web use
  </Accordion>
</AccordionGroup>
