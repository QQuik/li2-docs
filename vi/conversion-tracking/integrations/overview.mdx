## Chọn Integration Phù Hợp

<AccordionGroup>
  <Accordion title="Tôi muốn track conversions mà không động vào code">
    **Khuyến Nghị**: [Google Tag Manager](./google-tag-manager)

    GTM hoàn hảo nếu bạn:
    - Không có quyền developer access
    - Muốn kiểm soát tracking tự mình
    - Cần test trước khi deploy
    - Đã dùng GTM cho tools khác
  </Accordion>

  <Accordion title="Tôi chạy cửa hàng online (Shopify, WooCommerce)">
    **Sắp Ra Mắt**: E-commerce platform integrations

    Native integrations sẽ cung cấp:
    - Automatic order tracking
    - Không cần manual setup
    - Built-in e-commerce features
    - One-click installation

    **Giải Pháp Hiện Tại**: Dùng [Google Tag Manager](./google-tag-manager) hoặc [Direct SDK Installation](../setup)
  </Accordion>

  <Accordion title="Tôi cần kết nối Li2 với CRM/Marketing tools">
    **Sắp Ra Mắt**: Zapier và Segment integrations

    Chúng sẽ enable:
    - Automatic data syncing
    - Multi-platform workflows
    - No-code automation
    - Real-time updates

    **Giải Pháp Hiện Tại**: Dùng webhooks hoặc [API integration](../events) cho server-side tracking
  </Accordion>

  <Accordion title="Tôi có developers và muốn full control">
    **Khuyến Nghị**: [Direct SDK Installation](../setup)

    Direct integration cung cấp:
    - Maximum flexibility
    - Custom event tracking
    - Server-side tracking option
    - Full API access

    Bạn cũng có thể kết hợp direct SDK với GTM để có maximum control.
  </Accordion>
</AccordionGroup>
