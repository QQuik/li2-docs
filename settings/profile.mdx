# Profile Management

Manage your personal information, account security, and preferences through Li2.ai's comprehensive profile settings. Keep your account secure and personalized for optimal workflow efficiency.

## Accessing Profile Settings

<Steps>
  <Step title="Navigate to settings">
    1. Click on **Settings** in the main sidebar
    2. You'll see the Settings dashboard with multiple sections
    3. Profile-related options are organized under **Profile Settings**

       ![Accessing Profile Settings Pn](/images/Accessing-profile-settings.png)
  </Step>
  <Step title="Profile settings sections">
    Available profile management options:

    - **Edit Profile Information** - Personal details and contact information
    - **Update Password** - Account security and password management
    - **2FA Password** - Multi-factor authentication setup
  </Step>
</Steps>

## Account Security Management

### Password Management

<Steps>
  <Step title="Update your password">
    1. Navigate to **Update Password** section
    2. Enter your **Current Password** for verification
    3. Create a **New Password** following security best practices
    4. **Confirm Password** by re-entering the new password
    5. Click **Save** to update your account security
  </Step>
  <Step title="Password security best practices">
    **Strong Password Guidelines**:

    - Use a combination of uppercase and lowercase letters
    - Include numbers and special characters
    - Make passwords at least 12 characters long
    - Avoid common words or personal information
    - Use unique passwords for each account
  </Step>
</Steps>

### Multi-Factor Authentication (2FA)

<Note>
  **Enhanced Security**: Two-factor authentication adds an extra security layer by requiring a second verification method when logging into your account.
</Note>

**Authenticator App Setup**:

- **Purpose**: Adds extra security challenge during login
- **Recovery Option**: Email recovery available if you fail the authentication challenge
- **Recommendation**: Highly recommended for accounts with sensitive marketing data
- **Setup Process**: Configure through the **2FA Password** section in Profile Settings

  ![Multi Factor Authentication Pn](/images/Multi-factor-authentication.png)

**2FA Benefits**:

- **Account Protection**: Prevents unauthorized access even if password is compromised
- **Data Security**: Protects your touchpoint data and analytics information
- **Recovery Options**: Email-based account recovery maintains access if device is lost
- **Industry Standard**: Professional security practice for business accounts

## Organization Management

### Organization Settings

**Organization Profile**:

- **Name Organization**: Update your organization display name
- **Organization Updates**: Modify organization details and settings
- **Organization Management**: Access to organization-wide configuration

### Custom Domain Management

<Steps>
  <Step title="Add custom domains">
    1. Navigate to **Customize Domain** in Organization Settings
    2. Click **"\+ Add Domain"** to configure new custom domains
    3. Follow domain verification process
    4. Manage domain verification status and creation dates
  </Step>
  <Step title="Domain verification">
    **Domain Management Process**:

    - **Domain Registration**: Add your custom domain to Li2.ai
    - **Verification Status**: Track DNS verification progress
    - **Creation Tracking**: Monitor when domains were added
    - **Professional Branding**: Use your own domain for touchpoint URLs
  </Step>
</Steps>

### Organization Security

<Warning>
  **Danger Zone**: Organization deletion is permanent and cannot be undone. This action will remove all touchpoints, analytics data, and account information permanently.
</Warning>

**Organization Deletion**:

- **Permanent Action**: Cannot be reversed once completed
- **Complete Data Loss**: All touchpoints, analytics, and settings will be deleted
- **Account Termination**: Organization and all associated data permanently removed
- **Careful Consideration**: Only use when completely certain about account closure

<Steps>

    - Set up 2FA through the Authenticator App option

## Troubleshooting Profile Issues

<AccordionGroup>
  <Accordion title="Password and authentication issues">
    **Common Password Problems**:

    - **Password Reset**: Use email recovery if you forget your password
    - **2FA Problems**: Use email recovery option if authenticator app is unavailable
    - **Login Issues**: Verify email address and check for typos
    - **Account Lockout**: Contact support if unable to access account

    **Resolution Steps**:

    - Use "Forgot Password" functionality for password recovery
    - Check email (including spam) for recovery messages
    - Verify 2FA app is synchronized with correct time
    - Contact support for persistent access issues
  </Accordion>
  <Accordion title="Profile update issues">
    **Profile Update Problems**:

    - **Changes Not Saving**: Ensure all required fields are completed
    - **Email Update Issues**: Verify new email address format and accessibility
    - **Address Problems**: Check that address information is properly formatted
    - **Name Changes**: Ensure name updates meet character and format requirements

    **Troubleshooting Tips**:

    - Refresh page and try updating again
    - Clear browser cache if changes aren't persisting
    - Check internet connection for reliable form submission
    - Contact support if problems persist
  </Accordion>
</AccordionGroup>
